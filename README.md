# FOKUSIN: AI-Powered Time & Task Management System 🚀

> **Note:** This repository serves as a visual Front-End showcase. The core backend APIs, machine learning integration (Random Forest), and database architectures are securely maintained in a private team repository to protect proprietary source code.

## 📌 Project Overview
FOKUSIN is an advanced productivity web application integrated with AI for early detection of academic burnout. 

- **My Role:** Full-Stack & Front-End Developer
- **Tech Stack:** PHP (CodeIgniter), JavaScript, HTML/CSS, MySQL, FastAPI (Microservice)
